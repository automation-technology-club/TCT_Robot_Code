# LeRoy's automous robot car - GPS 

This is a semi-complexe robotic car, it uses GPS, a compass, and some other sensors to get to a waypoint. 

## Installation

Required Libraries:
NewPing.h
Adafruit_Sensor
Adafruit_HMC5883_U.h
waypointclass.h
softwareserial.h
math.h
moving_average.h
servo.h
TinyGPS.h
